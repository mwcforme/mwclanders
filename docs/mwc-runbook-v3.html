12. Incident Runbook

Site Down

1. Check UptimeRobot: https://dashboard.uptimerobot.com/monitors/803146043 2. Check Vercel deployments: https://vercel.com/men-s-wellness-centers/mwclanders/deployments 3. If bad deploy → Vercel → Deployments → previous → Redeploy 4. Check Sentry for JS crashes causing blank screen

Calendar Showing "Full" / Zero Slots

1. You will receive a slot-monitor email alert within 30 min if this persists 2. Trigger manual sync: POST https://lapvqhmmgeduuedleyhf.supabase.co/functions/v1/ghl-sync (Authorization: Bearer [anon key]) 3. Check GHL calendar availability is configured for that location 4. Supabase Dashboard → Edge Functions → ghl-sync → Logs for errors

WordPress Form Not Redirecting to Funnel

1. Check WP debug log for "FLUENT FORM - SUPABASE RESPONSE" 2. Response must contain funnel_url — if missing, see §9 above 3. Verify MWC_INTAKE_URL = https://lapvqhmmgeduuedleyhf.supabase.co/functions/v1/lead-intake 4. If correct URL but still no funnel_url, check Supabase → Edge Functions → lead-intake → Logs 5. Test with debug mode: https://book.menswellnesscenters.com/book/entry?t=TOKEN&debug=1

Leads Not Appearing in GHL

1. Check email — GHL failure alert fires immediately with lead details + manual action link 2. Check Supabase: lead_captures table → filter crm_status = 'failed' 3. Verify GHL API key not expired: [GHL-PIT-REDACTED] 4. Supabase → Edge Functions → lead-intake → Logs

No Lead Notification Emails

1. Check info subfolder at eobrien@menswellnesscenters.com first 2. Check SendGrid activity log for bounce or spam issues 3. Verify SENDGRID_API_KEY set in Supabase secrets 4. Supabase → Edge Functions → lead-intake → Logs for SendGrid errors

CI Pipeline Failing

1. GitHub → mwcforme/mwclanders → Actions → latest run → see which step failed 2. Common causes: - TypeScript error: npx tsc --noEmit locally to reproduce - Lint error: npm run lint -- --max-warnings 0 locally - Test failure: npm test -- --run locally - Bundle over 350 KB gzip: check dist/assets after npm run build - Secret scan: grep -rn "pit-|SG\.|vcp_|sbp_" src/ — remove any matches
